This is just a site for my projects, ongoing and old ones.

  • If you want a page like https://aviotron.github.io/newpage
  • first make new repo 'newpage' through github desktop
  • make new angular 2 project with name 'newpage'
  • copy the content of the project to the local git repository created in the previous steps
  • then run this command in cmd in the rot folder(make sure git is installed already) git remote add origin https://github.com/USERNAME/PROJECT_NAME.git
  • then run this commandgit push origin master
  • there is a utility gh-pages which let your publish github pages easily
  • run this to install it npm install -g gh-pages
  • now build the project using ng build --prod --base-href /PROJECT_NAME/
  • and finally to publish gh-pages -d dist
